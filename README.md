# Document-Clustering-
Applying clustering algorithms like Latent Dirichlet Allocation (LDA) or K-means to group similar documents together for topic modeling and understanding large text corpora. Resources : http://archive.ics.uci.edu/ml/datasets/Twenty+Newsgroups

# üß† Topic Modeling & Clustering on 20 Newsgroups
This project applies unsupervised machine learning techniques like K-Means clustering and Latent Dirichlet Allocation (LDA) to group and understand documents from the 20 Newsgroups dataset. It includes full preprocessing, modeling, and visualizations using tools like WordCloud, PCA, and pyLDAvis.

# üìÅ Dataset
Source: sklearn.datasets.fetch_20newsgroups

Data: 18,000+ newsgroup posts across 20 topics.

# üìå Features
‚úÖ Text preprocessing (cleaning, stopwords removal)

‚úÖ TF-IDF vectorization
